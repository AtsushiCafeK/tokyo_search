# データのメンテナンス方法

このプロジェクトでは、警視庁のオープンデータ（メールけいしちょう）を使用しています。
提供される生データに不備（末尾のHTML混入など）がある場合があるため、以下のコマンドを使用してデータをクリーンアップおよび配置してください。

## データ更新の手順

1. 最新の `mail_keishicho.json` を `import_json/` フォルダに配置します。
2. 以下のコマンドを実行します：

```bash
npm run update-data
```

このコマンドにより：
- `import_json/mail_keishicho.json` の破損がチェック・修復されます。
- クリーンなデータが `public/data/mail_keishicho.json` に配置されます。
- ブラウザ上のアプリが最新データを反映します。

## 補足
この処理はローカルで実行されるため、AIのトークンを消費することはありません。
コピー＆ペーストや手動での修復ミスを防ぐため、必ずこのツールを使用することをお勧めします。
